## the-last-dance

This app shows the user all of Michael Jordans championships and allows the user to upload their favorite memories.


## Technologies Used 
- HTML5
- CSS3
- Javascript
- Node.js
- Mongoose.js
- MongoDB
- Express.js
- Morgan
- EJS
- heroku
